# 基本规范
- 请正确使用全角中文标点符号。特别地，句末要有句号。

- 数学公式（运算式、运算符、参与运算的常数、作为变量的字母等）应使用 LaTeX，非数学公式（一般英文单词、题目名、算法名、人名等）不应使用 LaTeX。

- 中文与英文、数字或公式之间以半角空格隔开，但中文标点符号与英文、数字或公式之间不应有空格。

# 排版
- 应使用 markdown 正确排版。
- 应使用 #, ##, ###, #### 符号表示标题行。标题应对文章结构进行引导；不应滥用标题行表示强调与无意义内容。
- 应使用 -, +, * 来表示无序列表，用 1. 来表示有序列表。
- 应使用行内代码块表示字符串或代码，如 `aabc`。

- 应使用行间代码块引用代码。

- 应使用 []() 引用链接，如 题解审核及反馈要求（[题解审核及反馈要求](https://www.luogu.com.cn/discuss/174934)）。

- 应使用 ![]() 引用图片，如 ![](https://cdn.luogu.com.cn/upload/pic/22071.png)。

# 其余事项

* 标题是引导文章结构的，不是用来强调的。
  因此请不要用标题把字弄的很大，达到强调的目的。
* 加粗可以强调，但别大段。
* 代码请用 `` ` `` 包裹。
* 请正确使用项目列表来进行分点和步骤说明。

* 单词/公式和中文之间要加空格。
* 中文要用全角标点，末尾要句号。
  单词/公式和全角标点中无空格。
* 所有数学符号、常数、作为变量的字母应使用 LaTeX 公式。
* 行内字符串、代码片段用 `` ` `` 包裹。
* 不要把 LaTeX 公式滥用在非公式的单词或者缩写上。

* 所有数学符号、常数、作为变量的字母应使用 LaTeX 公式，用 `$` 包裹。
* 下标和上标的写法；公式和汉字中间加空格，但和标点中间不用。
* 大数字用科学计数法。
* 乘号要用 `\times` 而不是 `*`。小于等于号是 `\le` 而不是 `<=`。
* `log`/`min`/`mod`/`gcd` 等都有专门的表示。
