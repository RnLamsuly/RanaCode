# LinK16 放苹果
[题目链接](http://xmuoj.com/contest/362/problem/LinK16)

## 思路

使用[DP(动态规划)](https://oi-wiki.org/dp/basic/)可以简洁高效地解决这道问题。

对于每一个"分苹果"的问题，我们发现可以将其拆分成两种情况：
1. 有空盘子
2. 没有空盘子

假设现在将 $m$ 个苹果分入 $n$ 个盘子，且苹果比盘子多：
- 对于第 $1$ 种情况，我们可以先**空出 $1$ 个盘子**，然后再考虑往剩下的 $n-1$ 个盘子分苹果。
- 对于第 $2$ 种情况，我们可以先**拿出 $n$ 个苹果**，**每个盘子里先放一个苹果**以保证没有空盘，然后再分剩下的 $m-n$ 个苹果。

记循环变量 $i$ 为苹果数，$j$ 为盘子数，我们可以列出 $i > j$ 时的状态转移方程：

`dp[i][j] = dp[i][j-1] + dp[i-j][j]`

其中 `dp[i][j-1]` 是有空盘的情况， `dp[i-j][j]` 是没有空盘的情况。

此外，我们还需考虑**苹果少于盘子的情况**。如果苹果少于盘子，那么 $i$ 个苹果最多占据 $i$ 个盘子，我们可以忽略剩下的 $j - i$ 个盘子。即：

`dp[i][j] = dp[i][i]`

综合上面两种情况，我们给出最终的状态转移方程：

`dp[i][j]=(i>=j)?(dp[i][j-1]+dp[i-j][j]):dp[i][i]`

---

## 注意

初始化的时候要注意：

- **没有苹果**的情况对应的是 $1$ 种分配方案。

- 而**没有盘子**的情况被认为是 $0$ 种分配方案。

- **苹果与盘子均没有**的情况被认为是 $1$ 种分配方案。

这些初始条件是由状态转移方程与实际情况倒推出来的。

## AC代码
```cpp
#include <iostream>
using std::cin;
using std::cout;
int dp[15][15];
 
int main()
{
	int t;
	cin>>t;
	while(t--)
	{
		int n,m;
		cin>>m>>n;
		for(int i=0;i<=10;++i)
			dp[i][0]=0,dp[0][i]=1,dp[1][i]=1,dp[i][1]=1; //初始化
		for(int i=2;i<=m;++i)
			for(int j=2;j<=n;++j)
				dp[i][j]=(i>=j)?(dp[i][j-1]+dp[i-j][j]):dp[i][i]; //进行状态转移
		cout<<dp[m][n]<<'\n';
	}
	return 0;
}
```
