# LinK15 爬天梯
[题目链接](http://xmuoj.com/contest/362/problem/LinK15)

## 思路

这题的考察点实质上是递推，但题目要求我们使用递归实现。

我们用 $f(i)$ 表示走到第 $i$ 级台阶的方案数。注意到除了前两级台阶，走到第 $i$ 级台阶前一定经过的是第 $i - 1$ 级或者第 $i - 2$ 级台阶。 我们不难写出下面的递推式：

$$f(i) = f(i - 1) + f(i - 2) , n>2$$

所以我们将前两级台阶作为递归的终止条件，其余时候求 `f(i)` 的时候调用 `f(i-1)` 与 `f(i-2)`。

---

## 注意

如果你按照上述思路构建代码，**结果一定会TLE**。

由于题目要求我们使用递归，也就是每次求 `f(i)` 的时候都需要调用 `f(i-1)` 与 `f(i-2)` 。如果不加以优化，[时间复杂度](https://oi-wiki.org/basic/complexity/)将来到 $O(2 ^ n)$ 。

这里我们使用**记忆化搜索**：对于已经求过的结果，我们用一个记忆化数组存储起来。每次计算的时候，我们先检查记忆化数组有没有存储过相应的答案，如果有则**直接调用**，如果没有再**进行递归**。

---

## AC代码
```cpp
#include <iostream>
#define int long long
 
int a[50];
 
int fib(int n)
{
	if(a[n])return a[n]; //记忆化数组存储了结果，直接返回
	a[n]=fib(n-1)+fib(n-2); //记忆化数组没有结果，记录结果
	return a[n]; //返回结果
}
 
signed main()
{
	a[0]=1;
	a[1]=1; //前两级的方案数我们需要预处理
	int n;
	scanf("%lld",&n);
	printf("%lld",fib(n));
	return 0; 
}
```
