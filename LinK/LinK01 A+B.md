# LinK01 A+B
[题目链接](http://xmuoj.com/contest/362/problem/LinK01)

## 思路

这题常规思路是将两个输入的数直接求和输出。这里考虑一种不一样的思路：

我们知道位运算中的异或 (`XOR`) 是一种性质特殊的运算，即当两个二进制位不相同时得到 $1$ ，否则为 $0$ 。 因此，异或又被称做 “不进位加法” 。 而真实进行加法时，我们需要考虑进位，当且仅当某二进制位上的两个数都为 $1$ 的时候。 这与位运算中的与(`AND`)是一样的。这时候两个数 $\text{A}$ 与 $\text{B}$ 的和可以变为他们不进位加法的结果再加上进位的影响，即 `a + b = a ^ b + (a&b) << 1`。 用递归求和，当递归到 `a&b` 为0的时候即可将两数的和直接用异或求出。

---

## 注意
如何证明递归一定会终止，即 `a&b` 会在某一次递归中等于0 ?

## 1. 定义势能函数

设当前两个数为 $a$ 和 $b$，定义势能：

$$
\Phi(a, b) = \text{popcount}(a) + \text{popcount}(b)
$$

其中 $\text{popcount}(x)$ 表示 $x$ 的二进制表示中 **1 的个数**。

---

## 2. 递归公式

一次递归：

$$
a' = a \oplus b
$$

$$
b' = (a \& b) \ll 1
$$

---

## 3. 核心引理

**一次递归后，势能至少减少 1**（除非已经终止）。

即：

$$
\Phi(a', b') \le \Phi(a, b) - 1
$$

---

## 4. 证明引理

考察二进制**任意一位** $k$，设 $a_k$ 和 $b_k$ 分别是 $a$ 和 $b$ 在第 $k$ 位的值（0 或 1）。

分 4 种情况分析该位的 1 的个数变化：

| $a_k$ | $b_k$ | 原贡献 | 新贡献（$a'$ 和 $b'$ 在该位） | 变化 |
|-------|-------|--------|------------------------------|------|
| 0     | 0     | 0      | $a'_k = 0$，$b'_k = 0$       | **0** |
| 0     | 1     | 1      | $a'_k = 1$，$b'_k = 0$       | **0** |
| 1     | 0     | 1      | $a'_k = 1$，$b'_k = 0$       | **0** |
| 1     | 1     | 2      | $a'_k = 0$，$b'_{k+1} = 1$   | **-1** |

### 关键说明

当 $a_k = b_k = 1$ 时：

- 原本在第 $k$ 位有 **2 个 1**
- 递归后：
  - 第 $k$ 位：$a'_k = 0$（异或）
  - 第 $k+1$ 位：$b'_{k+1} = 1$（进位左移）
- 从 2 个 1 变成 1 个 1，**净减少 1**

---

## 5. 汇总所有位

对每一个二进制位 $k$：

- 如果 $a_k \ne b_k$：贡献不变
- 如果 $a_k = b_k = 0$：贡献不变
- 如果 $a_k = b_k = 1$：贡献 **减少 1**

因此：

$$
\Phi(a', b') = \Phi(a, b) - \text{popcount}(a \& b)
$$

---

## 6. 终止性证明

### 第一步：势能单调递减

只要 $a \& b \ne 0$，就有：

$$
\text{popcount}(a \& b) \ge 1
$$

所以：

$$
\Phi(a', b') \le \Phi(a, b) - 1
$$

势能严格减少。

### 第二步：势能有下界

$$
\Phi(a, b) = \text{popcount}(a) + \text{popcount}(b) \ge 0
$$

势能是非负整数。

### 第三步：有限步终止

因为势能每次至少减少 1，且不能小于 0，所以最多经过：

$$
\Phi(a_0, b_0)
$$

次递归后，必然有：

$$
a \& b = 0
$$

此时递归终止。

---

## 7. 完整结论

$$
\boxed{\text{递归必在有限步内终止，且步数最多不超过初始 popcount 之和}}
$$


> 最坏情况下时间复杂度为 $O(\log \text{N})$ ，即每次消除一个二进制上的进位。

## AC代码
```cpp
#include <iostream>
using namespace std;

// 递归加法：用异或和与运算模拟进位
inline int add(int a, int b)
{
    if ((a & b) == 0)          // 没有进位了
        return (a | b);        // 异或结果就是和（此时 a^b == a|b）
    return add(a ^ b, (a & b) << 1);
}

int main()
{
    int a, b;
    cin >> a >> b;
    cout << add(a, b) << endl;
    return 0;
}
```
