# LinK26 算24
[题目链接](http://xmuoj.com/contest/362/problem/LinK26)

## 思路

对于这题，我们考虑枚举法。枚举最终表达式的难度过高，且我们只需要判断是否有解，所以我们选择枚举算24点的过程。我们假设算24点的过程如下进行：

1. 从四个数字中先后选择**两个数字**。
2. 选择**一个运算符**。
3. 将第 $1$ 步选取的数字用第 $2$ 步选择的运算符进行运算，得到一个新的数字。
4. 在第 $1$ 步中未被选择的两个数字与第 $3$ 步新得到的数字，共计 $3$ **个数字**中先后选择 $2$ **个数字**。
5. 选择**一个运算符**。
6. 将第 $4$ 步选取的数字用第 $5$ 步选择的运算符进行运算，得到一个新的数字。
7. 决定第 $4$ 步中未被选择的数字与第 $6$ 步新得到的数字，共计 $2$ **个数字的先后**。
8. **选择一个运算符**，对第 $7$ 步中的数字运算。
9. 检验结果是否为 $24$。

这样，我们只需要枚举每一步中选取的数字或运算符，就可以遍历所有的情况了。总共需要进行 $8$ 次选择。

---

## 实现细节

为了统一每次操作后数字的顺序，我们规定每次未参与运算的数字按照索引顺序排在前，运算得到的新数字排在最后。

为了避免 $8$ 重循环，我们使用**状态压缩**。由于每次选择的选项个数不一样，我们的掩码需要使用**混合进位制** 。

掩码共八位：

- 第一位：表示第 $1$ 步中先选的数字( $0 \sim 3$ )
- 第二位：表示第 $1$ 步中后选的数字( $0 \sim 3$ )
- 第三位：表示第 $2$ 步中选择的运算符( $0 \sim 3$ )
- 第四位：表示第 $4$ 步中先选的数字( $0 \sim 2$ )
- 第五位：表示第 $4$ 步中后选的数字( $0 \sim 2$ )
- 第六位：表示第 $5$ 步中选择的运算符( $0 \sim 3$ )
- 第七位：表示第 $7$ 步中决定的先后数字( $0 \sim 1$ )
- 第八位：表示第 $8$ 步中选择的运算符（$0 \sim 3$）

状态共 $ 4 \times 4 \times 4 \times 3  \times 3  \times 4  \times 2  \times 4 = 18432$ 种。

---

## 注意

由于运算过程中涉及浮点数，比较两数是否相等的时候，我们应该如下方式比较：

```cpp
const double ERR = 1e-6;
...
if(fabs(a-b) < ERR)
...
```

此外我们要注意避免 $0$ 作除数的情况。

## AC代码
```cpp
#include <iostream>
int exe[8],num[4];
const int base[8]={4,4,4,3,3,4,2,4} ,MAX=18432;
const double ERR=1e-6;
 
bool equal(double a,double b) //针对浮点数的比较函数
{
	return (a-b<ERR)&&(b-a<ERR);
}
 
double calc(double a,double b,int op) //给定两个数与运算符编号，进行浮点运算
{
	switch(op)
	{
		case 0:return a+b;
		case 1:return a-b;
		case 2:return a*b;
		case 3:return equal(b,0)?114514:a/b;
	}
}
 
bool test()
{
	if(exe[0]==exe[1] || exe[3]==exe[4])return false;
	
	double a[3]; //a用来存储第一次运算过后的三个数字
	int pos=0;
	for(int i=0;i<4;++i)
	{
		if(i!=exe[0] && i!=exe[1])a[pos++]=1.*num[i];
	}
	a[2]=calc(num[exe[0]]*1.,num[exe[1]]*1.,exe[2]);
	if(a[2]>100000)return false;
	
	double b[2]; //b用来存储第二次运算过后的两个数字
	b[0]=a[3-exe[3]-exe[4]];
	b[1]=calc(a[exe[3]]*1.,a[exe[4]]*1.,exe[5]);
	if(b[1]>100000)return false;
	
	double c; //c是最终结果
	c=calc(b[1-exe[6]]*1.,b[exe[6]]*1.,exe[7]);
	if(equal(c,24))return true;
	return false;
}
 
int main()
{
	//freopen("9.in","r",stdin); 
	while(1)
	{
		for(int i=0;i<4;++i)scanf("%d",&num[i]);
		if(num[0]+num[1]+num[2]+num[3]==0)return 0;
		bool flag=0;
		for(int i=0;i<MAX;++i)
		{
			int t=i;
			for(int j=0;j<8;++j)exe[j]=t%base[j],t/=base[j]; //将掩码解压缩
			if(test()){flag=1;break;}
		}
		if(flag)printf("YES\n");
		else printf("NO\n");
	}
}
```
